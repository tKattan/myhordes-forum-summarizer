# Roadmap

## M1 - Extension foundation
Hello world extension, popup, content script injection, page validation

## M2 - Forum extraction
Thread list extraction, unread detection, post metadata extraction, normalized JSON output

## M3 - Incremental analysis
Local persistence for seen items, detection of only new content, debug output

## M4 - Prioritization
Rule-based scoring for pings, urgency, organization, and low-priority roleplay/chat

## M5 - Local AI summary
Local endpoint integration, prompt design, structured payloads, result parsing

## M6 - Usable popup
Simple result display, copy button, settings basics, better error states
